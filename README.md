### Project Overview
This project is a simple Python program that calculates the mean (average) and variance of a list of numbers entered by the user. It demonstrates fundamental statistical concepts and basic Python programming skills.


### Project Objective
The goal of this project is to:
- Understand how to work with user input in Python
- Perform basic statistical calculations
- Apply mathematical formulas using Python
- Practice writing clean, readable, and well-commented code



### Tools and Technologies
- Python 3
- Built-in functions (sum(), len())
- Basic Python concepts:
  - Lists
  - Loops (list comprehension)
  - Arithmetic operations


### How the Program Works
- The user enters numbers separated by commas
- The program converts the input into a list of numbers
- It calculates the mean (average)
- It calculates the variance using the mean
- The results are displayed clearly to the user
